# Receptenplanner

## Overzicht
en desktopapplicatie voor maaltijdplanning, met als uitgangspunt het slim combineren van recepten op basis van overlappende ingrediënten. 
Het project is opgezet met uitbreidbaarheid in gedachten, bijvoorbeeld richting planning op basis van budget of beschikbare kooktijd.

## Doel van het project
Dit project is opgezet als leer- en verdiepingsproject om mijn C#/.NET-kennis te actualiseren,
met specifieke aandacht voor architectuur, data modelling en onderhoudbaarheid.

## Functionaliteit
- Beheren van recepten en ingrediënten
- Weekplanning op basis van geselecteerde recepten
- Inzicht in overlappende ingrediënten

## Technische keuzes
- C# / .NET
- EF Core met SQLite
- Projectstructuur met duidelijke scheiding tussen data (EF Core), businesslogica (services) en UI
- Leesbaarheid en onderhoudbaarheid als leidraad bij ontwerp en naamgeving

## Status
Actief in ontwikkeling. Functionaliteit wordt iteratief uitgebreid en aangescherpt.

## Opmerking
Dit is geen productieapplicatie, maar een inhoudelijk leerproject waarbij keuzes en structuur
belangrijker zijn dan volledigheid.
